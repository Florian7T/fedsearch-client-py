# fedsearch-client-py
### Something to use until a proper way of using fedsearch premium is released :)

#### You can format your output using the following variables:
- {email}
- {password} or {hash}
- {ip}
- {token}
- {database}
- {count}

**Formatting examples:**
- {email} {password}
- {email}+{password} | {ip} {database}
- {count}: {email} ___ {hash}
